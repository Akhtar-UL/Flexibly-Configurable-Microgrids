# Flexibly-Configurable-Microgrids
ModularEMSGUI is the main application file. Launch this file to load and run the GUI.

Select the desired components by checking their corresponding boxes, then click Run Optimization to execute the optimization and view the results.

To modify the settings for a specific component, click the Configure button next to that component.

Before running the network simulations, unzip the OpenDSS_FeederLibrary folder and keep the folder in the same directory as other files. 

Select a test network from the drop-down menu located below the single-line diagram. Next, choose the bus where the microgrid will be connected, and then click Run QSTS to execute the network simulation.To view optimization and demand response (DR) results, select the desired graph from the Optimization Graph drop-down menu.

To view network and uncertainty (UNC/QSTS) results, select the desired graph from the UNC/QSTS Graph drop-down menu.
